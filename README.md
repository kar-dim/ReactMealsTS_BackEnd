# ReactMealsTS_BackEnd
My simple implementation (.NET/Java Backend) of Maximilian's "React Meals" (https://www.udemy.com/course/react-the-complete-guide-incl-redux/)

Uses NGROK for dev https hosting, EF Core (SQL Server db) and some basic Auth0 Authorization flaw (register/login, auth0 actions/claims/policies etc)

Branch springboot: backend uses Java Spring Framework and its technologies instead of .NET Core Framework
